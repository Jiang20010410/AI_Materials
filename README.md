# AI_Materials
This repository contains a collection of AI-driven projects for materials science, polymer informatics, and digital R&amp;D applications.
The projects focus on combining:

Materials science knowledge
Machine learning
Large Language Models (LLMs)
Data analytics
Chemical informatics
AI-assisted research workflows

The goal of this portfolio is to build practical AI tools for:

Polymer research
Materials property prediction
Scientific literature analysis
Structure–property relationship exploration
Digital laboratory workflows
